SETTINGメニュー　追加・変更リスト

基本的には原版のセッティングユーティリティと変わりませんが、
一部追加・変更・削除があります。
また、これより更に細かい設定はheboris.iniで可能です。

■追加・変更
・MAIN
　 NEXT PATTERN … 0(ランダム)8(MEMORY4式)9(7ピース1パック)10(電源パターン)11(電源別バージョン)が追加されました。
　 NEXT DISPLAY … デフォルトのNEXT表示個数を0〜6まで設定できます。
　 SHOW LEVEL 　… レベル表示の有無を設定できます。
　 MOVE SOUND　 … ブロック移動時の音の有無を設定できます。
　 BGM　　　　　… BGMファイルフォーマットを指定できます。（WAVE MP3 MIDI）
　　　　　　　　　 MIDIの場合は1曲のみです。その他はゲームモードごとにheboris.iniで設定できます。
　 WORLDREVERSE … WORLDルール時に回転方向を逆転させるかの設定です。
　 DOWN TYPE　　… 下を押しっぱなしにした時に、２ブロック目以降も
　　　　　　　　　 接着するのがNO RESET(HEBORIS)、しないのがDOWN RESETです。
　　　　　　　　　 20Gに突入した時、特に重要となります。
　 LVUP BONUS　 … ONにすると、3ライン消しの時4Lv、
　　　　　　　　　 4ライン消しの時6Lv上昇するようになります。

・DESIGN
　 BLOCK FRAME　… 操作中およびフィールド上のブロックの枠の有無を変更できます。
　 FIELD BG     … フィールド背景を変更します。
　 FIELD BG TR  … FIELD BGが「BLACK」の場合の透明度を設定します。
 　FONT＆DIGIT　… ルール別に文字色を設定できます。
　 NUMBER FONT　… SHARPに設定すると、一部の数字フォントが変化します。
　 MAX PLAYER　 … SINGLEに設定するとフィールドが真ん中に表示され、
　　　　　　　　　 2人プレイが出来なくなります。　動作が少し軽くなるかも？
　 BREAK EFFECT … ブロック消去エフェクトを表示するかの設定です。
　 SHOW COMBOS　… コンボ数を表示するかの設定です。
 　TOP FRAME  　… NEXTブロック周囲のフレームを2種類から選べます。

・INPUT
 　Dボタン（HOLD）が追加されました。
　 また、パッド用の設定も（1P、2P）可能です。
　 [FUNCTION KEY]では、既存の1P・2Pのポーズ・NEXT隠しのほかに、
　 捨てゲーボタンとスクリーンショットボタンが設定できます。
