★ はじめに ------------------------------------------------------------------

    　Heboris [ヘボリス] 2001をダウンロードいただき、ありがとうございました。
    Heboris 2001 は、どこかで見たことがあるような!? 落ちものパズルゲームです。
　　このゲームはフリーソフトとなっています。
      また、インターネットランキングも開催していますので、ふるってご参加くださ
    い。こちら → http://www.hosiken.afis.to/iranking/heboris/

    　ダウンロードしたアーカイブの中身を、適当な場所に解凍してください。

★ 実行方法 ------------------------------------------------------------------

      エクスプローラ等から、 Heboris.exe をそのまま実行してください。
      終了は ESC キーです。押せばいつでも終了するはずです。

★ 操作方法 ------------------------------------------------------------------

      キーボード・ジョイスティックに対応しています。
      標準のキーボードのキー配置は、以下の通りです。

      主なキーのキーボードの配置は、同梱の「セッティング ユーティリティー」から
      カスタマイズが可能です。(NEW! V1.50)

      ■ ゲーム終了 : ESC

      ■   F1    :  1P ポーズ (一時停止)
      ■   F2    :  2P ポーズ (一時停止)
      ■   F3    :  1P の NEXT ブロックを隠す (または、表示する)
      ■   F4    :  2P の NEXT ブロックを隠す (または、表示する)

      ■ フルキーの 1, 2, 3, 4, 5, 6, 7, 8, 9, 0 :
                    ゲームオーバー時に押すと、リプレイを保存します (1Pのみ)
                    左からファイル No. 01 〜 10 に対応しています
      ■ フルキーの Q, W, E, R, T, Y, U, I, O, P :
                    ゲームオーバー時に押すと、リプレイを保存します (2Pのみ)
                    左からファイル No. 11 〜 20 に対応しています

      ■   上    :  1P: S           2P: カーソルキー・↑
      ■   下    :  1P: X           2P: カーソルキー・↓
      ■   左    :  1P: Z           2P: カーソルキー・←
      ■   右    :  1P: C           2P: カーソルキー・→

      ■ ボタン1 :  1P: B           2P: テンキー・1
      ■ ボタン2 :  1P: N           2P: テンキー・2
      ■ ボタン3 :  1P: SPACE       2P: テンキー・3

      ※ ボタン1... 回転 & ゲームスタート
      ※ ボタン2... 逆回転
      ※ ボタン3... 回転

★ 遊び方 --------------------------------------------------------------------

      上から、4つの石で構成された、ブロックが落ちてきます。このブロックは、
    ボタン1〜3で回転、左右で移動、下で落下、上で高速落下 (セッティングユーティ
    リティーで使用許可した場合のみ) というように動きます。
      このブロックが、地面に接触すると、それは固定され、また次の新しいブロック
    が落ちてきます。

      このブロックを、うまく隙間なく積み上げて、横一列のラインを作ってください。
    うまくラインを作ると、そのラインは消滅し、得点となります。一度にたくさんの
    ラインを消すと高得点です。
      連続でラインを消しても、「コンボ」となり、ちょっとだけ高得点になります。

      下や上を押して落下させても、わずかながら点数が入ります。下 (落下) よりも
    上 (高速落下) のほうが、加算される点数が微妙に高くなります。
      全てのブロックを消すと、「全消し」となり、そのときにラインを消した獲得得
    点が、通常の 10 倍になります。

      ブロックが上まで積みあがると、ゲームオーバーです。

      ゲームを続けていくと、レベルが上がり、自然落下速度がだんだん速くなります。
    最終的には、空中をワープする程速くなり、なかなか手に負えなくなります。
      そこで、次のブロックが出現するまでに、ボタン1〜3のいずれかを押しつづけると、
    既に回転 (または逆回転) した状態でブロックが出現します。これは後半のレベルで
    は重要なので、よく覚えておいてください。

                                      ●

      タイトル画面では、「SOLO MODE」「VERSUS MODE」「REPLAY」「RANKING」が選択
    できます。

      ■ SOLO MODE   … ひとりでプレイするモードです。ふたりのプレイヤーが
         それぞれ独立してゲームをすることもできます。
      ■ VERSUS MODE … ふたりで対戦するモードです。
      ■ REPLAY      … SOLO MODE で記録したプレイ内容を再生できます。
      ■ RANKING     … SOLO MODE のスコアランキングを表示します。

                                      ●

      SOLO MODE を選択すると、難易度の選択ができます。

      □ BEGINNER    … 比較的難易度の低いモードです。レベルアップもゆっくり
         で、じっくり遊べます。
      □ MASTER      … 比較的難易度の高いモードです。さっくり遊べます。あな
         たの腕によっては、恐怖の最高速に挑むことも可能です。
      □ 20G MODE    … ゲームをスタートした瞬間から最高速で始まる、上級者向け
         モードです。
      □ DEVIL       … 20G MODEをも超える難易度の、超上級者向けモードです。

                                      ●

      このモードでは、えんえんとブロックを消していきます。ゲームオーバーになる
    まで続きます。
      空いてる側のプレイヤーは、ボタン1を押すことにより、いつでも途中参加がで
    きます。それぞれのプレイヤーは、完全に独立してゲームが進行します。

      両者ゲームオーバーになった状態で約1秒が経過すると、タイトル画面に戻りま
    す。このモードでは、両方のプレイヤーがゲームオーバーにならない限り、毎ゲー
    ム、同じ順番でブロックが出現します。

                                      ●

      SOLO MODE で「GAME OVER」の文字が表示されている間に、以下のキーを押すと
    今回のゲームのプレイ内容をセーブすることが出来ます。このプレイ内容は、あと
    で「REPLAY」から再生して、見ることが出来ます。

    ■ 1P サイドにのみに有効なキー
       1, 2, 3, 4, 5, 6, 7, 8, 9, 0 (左から、ファイル No.01 〜 10 に対応)
    ■ 2P サイドにのみに有効なキー
       Q, W, E, R, T, Y, U, I, O, P (左から、ファイル No.11 〜 20 に対応)

      たとえば、1P がゲームオーバー中に、3 キーを押すと、ファイル No.03 に、今回
    の 1P のプレイ内容を保存することが出来ます。
      文字が表示中にボタン1を押すと、ゲームオーバー画面がスキップされますので、
    ご注意ください。

    ※ 20分を超えるプレイデータは保存できません。あらかじめご了承ください。

                                      ●

      VERSUS MODE を選択すると、双方のプレイヤーの難易度 (落下速度) を決定し
    ます。難易度は、一応 5 段階ということになっていますが、さらに下には…。

      このモードでは、ふたりのプレイヤーで対戦ができます。一度に2列以上消すと、
    相手のフィールドをせり上げることができます。どちらか片方のプレイヤーがゲ
    ームオーバーになると、ゲーム終了。もう片方のプレイヤーが勝者となります。

      難易度選択画面で、どちらかのプレイヤーのボタン2を押すと、タイトル画面に
    戻ります。

                                      ●

      REPLAY を選択すると、SOLO MODE で記録したプレイ内容を再生することが出来
    ます。リプレイ再生中は、以下の操作が出来ます。

      ■   上    :  早送り (2 倍速)
      ■   下    :  早送り (4 倍速)

      ■ ボタン1 :  早送り (6 倍速)
      ■ ボタン2 :  タイトル画面に戻る

                                      ●

      RANKING を選択すると、SOLO MODE のスコアランキングを表示します。
      この画面では、以下の操作が出来ます。

      ■  左/右    : ゲームモードの切り替え
      ■ ボタン1/2 : タイトル画面に戻る

★ HIDDEN MODE について (NEW! V1.50) -----------------------------------------

      SOLO MODE または VERSUS MODE の難易度選択画面で、右方向キーを数回押すと
    恐怖の「HIDDEN MODE」に挑むことができます。左方向キーでキャンセルできます。
      このモードは、時間の経過とともに、積み上げたブロックが次第に見えなくなっ
    てゆく恐ろしいモードです。右方向キーを押した回数により、レベルを選択するこ
    とができ、そのレベルによって、見えなくなるまでのスピード、見えなくなる最大
    段数が異なっています。

      見えなくなってしまったブロックは、消したラインだけもとに戻すことができま
    す。

★ FAQ -----------------------------------------------------------------------

      Q.「コンパイルエラーです」と表示され、ゲームを起動できません。

      A. Heboris.exe と同じフォルダに、 Option.def と KeyConf.def いうファイル
         が入っているか確認してください。ある場合は、 これらのファイル の中身が
         壊れている可能性があります。その場合は、それらファイルを一旦削除し作り
         直してください。
         また、ない場合も、同様にセッティングユーティリティーから新たに作成し
         てください。

         Option.def は、セッティングユーティリティー(Setting.exe)で「設定を保
         存」すると、作成できます。
         KeyConf.def は、セッティングユーティリティー(Setting.exe)の「キーボー
         ドの設定」で「設定を保存」すると、作成できます。

         ※ セッティングユーティリティーは、必ず Heboris 本体のバージョンに対
            応したものを使用してください。
         ※ エラー時に生成される CompileError.txt は不要ですので削除してください。

                                      ●

      Q. ゲームの動作がすごく遅いのですが…。

      A. [解決策1] セッティングユーティリティーで、サーフェス格納場所を「VRAM」
           に設定してください。
         [解決策2] ウインドウモードで起動する場合は、Windows の「画面のプロパ
           ティ」の」画面の色数を「16 ビットカラー」に設定してください。
         [解決策3] グラフィックボードの種類によっては、「ウインドウモード (640
           ×480)」の動作がすごく遅い可能性があります。その他のスクリーンモード
           で起動してください。

         これでも解決しない場合は、パソコンのスペック不足などが考えられます。

                                      ●

      Q. キーボードのキーの割り当ては変更できますか?

      A. V1.50 からできるようになりました。セッティング ユーティリティーから、
         「キーボードの設定」ボタンをクリックしてください。

                                      ●

      Q. ジョイパッボタンキーの割り当ては変更できますか?

      A. 単体ではできません。JoyAddapter を使用すると便利です。JoyAdapter は
         K2 さん (http://okamoto.ws/cc/) が作成されたフリーソフトです。

                                      ●

      Q. インターネットランキングの詳細を知りたいのですが。

      A. 以下のサイトをご覧ください。
         http://www.hosiken.afis.to/iranking/heboris/

                                      ●

      Q. ランキングデータはどこに保存されているのですか。

      A. Heboris.exe と同じフォルダに、Ranking.sav というファイルに保存されて
    います。このファイルを消せば、ランキングの内容を初期化できます。

                                      ●

      Q. リプレイデータはどこに保存されているのですか。

      A. Heboris.exe と同じフォルダに、ReplayXX.sav というファイルに保存されて
    います(XX の部分はファイル No. に置き換わります)。このファイルを消せば、ラ
    ンキングの内容を初期化できます。

★ 特殊機能 ------------------------------------------------------------------

      Heboris.exe と同じフォルダに、 BLOCKSE というフォルダを作成し、中に
    Block1.wav 〜 Block7.wav という名前の、7個の WAVE ファイルを設置すると、
    NEXTブロック登場時に音が鳴ります。ただそれだけの機能です。

      どの音が鳴るかは、次のNEXTブロックによって決まります。

      → 1.赤(ヘボリス棒) 2.橙(L字形) 3.黄(正方形) 4.緑(Z字形)
         5.水(T字形) 6.青(逆L字形) 7.紫(逆Z字形)
