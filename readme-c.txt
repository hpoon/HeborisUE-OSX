【インストール方法】
解凍したファイルのうち

replayフォルダ（中身は空）
stageフォルダ（中身は空）
ssフォルダ（中身は空）
capフォルダ（中身は空）
scriptフォルダ
spiフォルダ
speed.def
graphics.dat
graphicshi.dat
stage.dat
tittle_mov.dat
heboris.ini
heboris_user.ini
HEBORIS.EXE [new!]
compiling.ini [new!]

を、ヘボリスのフォルダにコピーします。
（上書き確認が出た場合は「heboris_user.ini」以外を上書きしてください）

【重要】
・今までセッティングユーティリティで行っていた設定は
　タイトル内の「SETTING」からでしか行えません。（設定内容はsetting.txtにて）

・Windows XP以降のWindows OSで動かすには、以下の手順で互換モードを有効にしてください。
　１．HEBORIS.EXEを右クリックし、[ショートカットの作成] をクリックします。
　２．作成されたショートカットを右クリックし、[プロパティ] をクリックします。
　２．[互換性] タブの [互換モードでこのプログラムを実行する] をクリックし
　　　「Windows NT 4」以外のいずれかのオプションをクリックします。
　３．OKボタンをクリックします。

【オリジナル版からの追加・変更事項】
　●高解像度（640x480）対応
　　　ゲーム内のSETTINGの、SCREEN MODEで「FULL SCREEN 640X480」または「WINDOW 640X480」（及びそれより高い解像度）を
　　　選択すると、綺麗なグラフィックでプレイすることが出来ます。

　●ゲームモード
　　・TOMOYOモード
　　　 フィールド上に配置されたプラチナブロックを時間内に全て消せばステージクリアとなるモードです。
　　　 制限時間にはステージごとのタイム（１面あたり１分）と、全体を通してのリミットタイム（ゲーム開始時３分）があります。
　　　 ステージタイムがなくなってもゲームオーバーにはならず、次の面がスタートします。
　　　 リミットタイムがなくなるとゲームオーバーです。
　　　 リミットタイムはプラチナブロック消去かステージクリア時の成績で回復します。
	(TOMOYO-TI)全20面で、成績次第ではさらにEXステージをプレイできます。
	(TOMOYO-EH)全18面で、ステージごとのタイムは無制限です。リミットタイムは10分です。
	(ACE-TGT)　全21面で、ステージごとのタイムは無制限です。
	　　　　　　ただしプラチナブロックがランダムで配置されます。
	(TOMOYO-Edit)STAGE67〜99の、自作ステージをプレイします。ステージごとのタイムは無制限です。
	　　　　　　　事前にステージデータを作成しておく必要があります。
	(F-Point) [18MIN]上記4モードとはゲーム性が若干異なります。
	　　　　　　最初に1〜90のステージを選択します。選択したステージをクリアすると、
	　　　　　　下部に表示されているBONUSがスコアに加わります。
	　　　　　　そして次のステージを4つの中から選択します。
	　　　　　　途中、ステージ90をクリアしないと、その先のステージへは進めません。
	　　　　　　ステージ100をクリアすればオールクリアとなり、10万点のボーナスが入ります。
　　　　　　　　　　リミットタイムは18分ですが、回復させる手段はありません。
 		　[BASIC]基本的には18MINモードと同じですがリミットタイムはありません。
			ただしリプレイも録れません。レベルアップ方式がHEBO方式となっており4列または
			一定数のブロックを落とすとレベルアップします。レベル7以内でステージクリアすると
			レベルが一つ下がります。7以上でクリアすると、6に戻されます。
　　・ACEモード
　　　 SOLO MODEの中に追加されました。
　　　 制限時間内にラインを消していくモードです。
　　　 レベルアップ方式が独特のもので、ブロックを置くだけではレベルアップは発生しません。
　　　 指定ライン消すたびにレベルが1つ上昇し、制限時間が回復します。
	ACE
　　　　　10ライン消すごとにレベルアップし
	　合計で150ライン消去するとクリアとなります。
	ANOTHER
　　　　　ACEよりも落下速度が速く、制限時間も短めなモードです。
	HELL
　　　　　初めからANOTHERの最高速です。200ラインでクリアですが…？
	OLD-STYLE
	　4ライン消すごとにレベルアップするACEのエンドレスモードです。

　　・PRACTICEモード
　　　 タイトル画面から選べます。
　　　 落下速度・固定までの時間など、様々な設定を変更してプレイすることが出来ます。
　　　 
　　　 なお、SETTYAKUの値を99に設定すると、固定までの時間が無限になります。20Gの練習にオススメです。

　　・MISSIONモード
　　　 タイトル画面から選べます。
　　　 指定された課題を次々とクリアしていくモードです。
　　　 自分で課題を作ることも出来ます。（mission.txt参照）

　　・VERSUSモード　　リプレイが記録できるようになりました！ ←[new!]
　　　 コンピュータとの対戦が可能になりました。（デフォルトでは2P側がCPUです。）
　　　 コンピュータの有無や性格はheboris.iniで設定できます。
　　　 様々な機能が追加されています。(\doc\versus.txt参照)
　　　 更に、せり上がりのパターンを4種（ランダム･自フィールド最下列コピー･HEBO+のパターン･消した形を上下反転）から選べます。

　　・SIMPLEモード
　　　このモードではレベルアップによるスピードの変化はありません。
　　　40LINESとULTRA2minのスピードは開始前に設定できます。
　　　40LINES：40列消すまでのタイムを競うモード
　　　ULTRA2min：2分間で何列消せるかを競うモード
      SQUARE：4x4の正方形を作って消すモード
      MARATHON：レベル15を目指すモード

		ROT.RELAYモード
		 40LINESかULTRA2minを全ての回転法則でプレイします。
		 窒息してもゲームオーバーにはなりませんが、ペナルティが加算されます。
		 プレイする回転法則の順番は自由に選べますが、速度は開始前に設定した値で固定です。

　　・SPECIALモード
	BIG MODE
	　Cボタン押しっぱなしでBEGINNER、MASTER、20G、DEVIL、ACEモードのいずれかを
	　開始すると、出現するブロックが常に2倍の大きさになります。
	BIGINNERのSCORE TRIAL
	　ビギナーモードにカーソルを合わせてHOLDボタンを押すと、
	　花火の数ではなく、スコアを競うモードになります。
	　レベル100と200の手前ではレベルストップが無く、
	　決められたお助けアイテムが出現します。レベル300でクリアです。
	　早くレベル300に到達するとタイムボーナスが入ります。
	　コンボよりもBACK to BACKの方が高得点が狙えるでしょう…
	ITEM MODE	
	　マスター、20G、デビルモード[new!]にカーソルを合わせてHOLDボタンを押す（デビルは2回押す）と、
	　アイテムモードが選択できます。対戦と同じように一定個数ブロックを置くと
	　アイテムが出現しますが、攻撃アイテムの効果は自分に来ます。
	HEBO+
	　マスターまたは20Gモードにカーソルを合わせてHOLDボタンを２回押すと、
	　HEBO+モードが選択できます。たまに下から灰色のおじゃまラインがせり上がってくるモードです。
	　序盤の落下速度は各モードと同じですが、LV500以降は全く変化しなくなります。
　　　　　ただし段位4設定時はいつものように加速していきます。
　　　 　 LV999でクリアですが、スタッフロール中でもせり上がりが発生します。	　
	EXAMINATION
	　MASTER、20Gを普通にプレイしようとした時、一定の確率でフィールドが震えだして…？
	DEVIL-MINUS
	　通常のDEVILの速度が速すぎると感じた方は試してみてください。
	　DEVILモードにカーソルを合わせて右移動入力を行うと選べます。
	　レベル1000でクリアです。
	DEATH+
	　DEVILモードにカーソルを合わせてHOLDボタンを押すと選択できます。
	　NEXTブロックは1つしか見えず、HOLDも使用できない状態で
	　DEVILモードをプレイできます。
　　　　（DEVIL-とDEATH+は併用する事が出来ます。）

　●システム
　　・HOLD
　　　 画面左上のHOLD欄にブロックを保留できる機能です。
　　　 保留したブロックは操作中のブロックと入れ替えることができますが、
　　　 一度入れ替えたブロックは、次のブロックが出現するまでHOLD欄に入れることはできません。

　　・回転法則選択
　　　 今までのルール（HEBORISルール）に加え、新たに7つのルールが選べるようになりました。
　　　　・TI-ARSルール
　　　　　　基本はHEBORISルールと一緒ですが、水色(T字)・赤(棒)の回転がより柔軟になりました。
　　　　・TI-WORLDルール
　　　　　　HEBORIS・TI-ARSルールとは大きく異なります。
　　　　　　壁蹴り・床蹴りが起こりやすく、操作次第ではブロックを駆け上がることも可能です。
　　　　　　また、下入力ではブロックを接着させることができず、上を入れたとき瞬間落下と同時に接着されます。
　　　　　　回転したり移動したりするだけで固定されるまでの時間が延長されるので、難易度は低めです。
　　　　　　なお、デフォルトでは「WORLD」および「SRS」とつくものは回転方向が逆ですが、
	　　SETTINGメニューの「WORLDREVERSE」で変更できます。
　　　　・ACE-SRSルール
　　　　　　基本はTI-WORLDルールと一緒ですが、回転ボタンを連打しているとなかなか接着されません。
　　　　　　さらに、高速落下の速度が落ちています。初心者に最適です。
　　　　・ACE-ARSルール
　　　　　　回転法則だけがTI-ARSルールで、あとはACE-SRSルールという奇抜なルールです。
　　　　・ACE-ARS2ルール
　　　　　　ACE-ARSルールの上下の操作をTI-ARSルール形式にした以外に違いはありません。
　　　　　　ARS系列の練習に最適です。
　　　　・DS-WORLDルール
　　　　　　基本はACE-SRSルールと一緒ですが、回転ボタンを連打している限り永遠に固定されません。
　　　　　　また、ブロックを置いたあとに待ち時間なしに次のブロックが出現します。
　　　　・SRS-Xルール
　　　　　　基本はACE-SRSルールと一緒ですが、上下入力がTI-ARSと同じになっています。
　　　　　　また、Cボタンは「180度回転」となっており、
　　　　　　使い方次第ではとんでもない所にブロックをねじ込む事も可能です。
	・D.R.Sルール
	　　TI-ARSをベースにしながら、強力な回転入れや手動AREスキップなどの
	　　独自の要素を多数搭載しています。

　　・TGM式レベルアップ追加
　　　 ブロック出現・ライン消去でレベルが上がります。
　　　 設定次第で３・４列消しでボーナスが付くようにも出来ます。

　　・新先行回転入力
　　　 ブロックの先行回転入力が3種類になりました。
　　　 heboris.iniで変更できます。（デフォルトはオリジナル版と同じです。）

　　・メダル表示
　　　 プレイの状況に応じて５種のメダルを表示します。（AC・SK・CO・RE・ST）
　　　 また、同じ種類のメダルでもプラチナ・金・銀・銅の４色があります。

　　・タイムトライアルモード＆スタッフロール
　　　 改造版ヘボリスの基本は非エンドレスのタイムアタックです。
　　　 BEGINNERはLv200、MASTER・20GはLv999、DEVIL-DOOMはLv1300、DEVIL-MINUSはLv1000に到達するとクリアですが、
　　　 BEGINNER以外のモードでは規定タイムを突破しないとLv500（DEVIL-DOOMのみLv1000でも）で強制終了です。
       なお、規定タイムはheboris.iniで設定可能です。
　　　 それを乗り越え規定Lvに到達するとスタッフロールが始まり、
　　　 1分間耐え切ると完全クリアとなります。
　　　 BEGINNER-SCORE TRIALとDEVIL-MINUSは約30秒で、途中で窒息しても完全クリアになります。

　　・新ランキング
　　　 heboris.iniのranking_typeで、2タイプのランキング記録方式が選べます。
　　　　0の場合→全回転法則のランキングで、BEGINNER･MASTER･20G･DEVILの4モード
　　　　　　　　　　　　（それぞれエンドレスとタイムトライアルの2タイプ）の記録が残ります。
		　　　　 順位の基準はエンドレスがスコア優先で、タイムトライアルだとタイムが優先されます。
		　　　　 タイムトライアルの場合、同じタイムでもスタッフロールをクリアしている方が順位は上になります。

	1の場合→ランキングが回転法則グループ別に分かれていて、
		　　　　 BEGINNER･MASTER(段位タイプ別)･20G･DEVIL･TOMOYO(TIコースかEHコースのみ)･ACE(通常及びANOTHER)･エンドレス（全モード）の記録が残ります。
		　　　　 順位の基準はモードごとに異なります。
　　　　2の場合→ランキングが回転法則グループ別に分かれていて、
		　　　　 BEGINNER･MASTER(段位タイプ別)･20G･DEVIL･TOMOYO(TIコースかEHコースのみ)･ACE(通常及びANOTHER)･エンドレス（全モード）の記録が残ります。
		　　　　 順位の基準はモードごとに異なります。

　　・一人用でのせり上がり

　　　 せり上がるラインの形状や、せり上がり間隔には法則性があります。

【軽くするには？】
■ゲーム内の「SETTING」
・解像度を「320x240(FULL SCREEN)」にする
・COLOR MODEを「HIGH COLOR (16BIT)」にする
・MAX PLAYERをSINGLEにする

■heboris.iniの項目
・「breaktype」を「2（旧TAP/TI）」にする
・skip_viewbgを「1」にして背景を非表示にする
・skip_objを「1」にしてエフェクトを無効にする
・disable_giji3Dを「4」にして擬似立体表示を無効にする
・alpha_tlsを「0」にしてTLSを不透明にする

【heboris_uer.iniの使い方】
heboris.iniの項目を更新毎に変更するのは面倒だ、という人は
該当項目を「heboris_user.ini」に書き込んでおき、
更新時にheboris_user.iniを上書きしないようにすれば
設定を引き継げます。
